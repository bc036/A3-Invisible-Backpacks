# A3-Invisible-Backpacks
Adds invisible backpacks to Arma 3. 

## Includes:
- Invisible Assualt Pack 
- Invisible Bergen 
- Invisible Carryall 
- Invisible Kitbag 
- Invisible Field Pack 
- Invisible Tactical Pack
- Invisible Parachute
- Invisible Viper Harness
- Invisible Viper Light Harness
